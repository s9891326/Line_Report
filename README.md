### 操作步驟
1. 到此網址進行登入 https://notify-bot.line.me/zh_TW/
![image](resource/1.jpg)

2. 登入後點選右上角的圖示，再點擊「個人頁面」
![image](resource/2.jpg)

3. 進入後往下拉，會看到按下「發行權杖」
![image](resource/3.jpg)

4. 接下來必須設定權杖通知、通知哪個群組，按下發行後，會有一組token產生，必須記起來這組token碼
![image](resource/4.jpg)
